# Preston Auto Care Ltd Website 🚗🔧

A modern, responsive, and highly optimized commercial website built from scratch for an active automotive service garage. This project served as a comprehensive, real-world modernization effort—replacing a legacy, high-cost system with a high-performance frontend and a fully automated DevOps deployment pipeline.

🔗 **[Launch Live Website](https://prestonautocareltd.pages.dev)**

---

## 💼 Business Case & Problem Solved

The business was previously relying on an outdated legacy website built and hosted back in 2017 via IONOS. Upon conducting a technical audit of the client's infrastructure, two major pain points were identified:
1.  **Financial Inefficiency:** The client was paying an inflated hosting package fee of £180 annually.
2.  **Poor User Experience:** The existing site was uninformative, non-responsive, and lacked modern design practices, severely limiting visitor conversion.

### The Solution:
I completely redesigned and engineered a modern web presence from the ground up, providing a custom logo design to elevate brand visuals, structuring highly informative layouts to attract new local customers, and **reducing the client's recurring hosting overhead to £0/year** by leveraging modern cloud network architectures.

---

## 🛠️ Tech Stack & Implementation Details

### Frontend Architecture
*   **Structure:** Semantic HTML5 to ensure strong local SEO foundation and accessibility.
*   **Layout Engine:** Modern CSS3 Grid and Flexbox for precise, fluid responsive design across all mobile and desktop screen sizes.
*   **Design:** Custom logo and visual overhaul tailored to modern automotive branding.

### DevOps & Infrastructure
*   **CI/CD Pipeline:** Fully automated deployments configured via **GitHub Actions**. Pushing code updates to the main branch automatically triggers a build and deploy sequence.
*   **Domain & Infrastructure Management:** Integrated with **Cloudflare DNS** to ensure enterprise-grade security, rapid edge caching, global asset delivery, and maximum uptime.
*   **Hosting Network:** Leveraged continuous cloud deployment networks to completely eliminate legacy hosting fees for the client.
